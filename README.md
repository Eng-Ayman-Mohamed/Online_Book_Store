## Project Overview

The **Online Book Store** is a full-stack web application designed to manage book sales and orders efficiently. The system supports two types of users: **Customers** and **Managers**, each with specific functionalities.  

---

## Features

### User Roles

#### Customer
- Browse available books  
- Place orders for books  
- Add books for sale  
- View orders in charts  

#### Manager
- Promote customers to higher privileges  
- Add new books to the store  
- Modify or remove existing books  
- Perform all customer actions  
- Confirm orders  

---

## Problem Specifications

- Two user types: **Customer** and **Manager**  
- Customers can buy and sell books  
- Managers oversee operations, manage users, and confirm orders  
- Most core functionalities have been implemented

## Getting Started

1. Clone the repository:  
   ```bash
   git clone https://github.com/Eng-Ayman-Mohamed/Online_Book_Store.git

2. Set up the MySQL database using the provided schema.
3. Run the Spring Boot backend:

   ```bash
   mvn spring-boot:run
   ```
4. Navigate to the frontend directory and start React:

   ```bash
   npm install
   npm start
   ```
5. Open your browser at `http://localhost:3000` to access the application.
